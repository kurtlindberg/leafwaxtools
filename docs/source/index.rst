leafwaxtools has been made freely available under the terms of the `Apache 2.0 License <https://github.com/kurtlindberg/leafwaxtools/blob/main/LICENSE>`_. There are many ways to get involved :ref:`with the development of leafwaxtools<contributing_to_leafwaxtools>`:

* Report bugs and problems with the code or documentation to our `GitHub repository <https://github.com/kurtlindberg/leafwaxtools/issues>`_.

  * Contribute enhancements and new features
  * Contribute to the code documentation
